# Python_work
AotuTest_Script 是基于python+selenium编写的UI自动化脚本，涉及以下常见场景及对应的处理方法：

***
author：huangzy
date：2020/03/12
***

1、select形式的下拉框及DOM结构形式的下拉框定位方法；
2、控制滚动条滚动到页面指定位置；
3、使用第三方工具实现调用本地文件并上传文件；
4、无法定位元素唯一属性（或动态变化元素）时的处理方法（使用for循环在变化区间内遍历出现的元素）；
5、配置日志文件，打印日志信息；
6、套用测试框架，规范测试流程及用例；
7、引用第三方邮件插件，实现自动发送测试报告邮件；
8、自定义测试报告文件，自动生成测试报告；
9、操作弹窗处理方法。
